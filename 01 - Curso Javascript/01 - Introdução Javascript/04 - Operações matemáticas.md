# Operações matemáticas

Agora, realizaremos algumas operações matemáticas.

```javascript
var x = 10;
var y = 5;

var sum = x + y; // 15
var sub = x - y; // 5
var mul = x * y; // 50
var div = x / y; // 2
var mod = x % y; // 0

x = 10;
y = 5;

x += 3; // 13
y -= 3; // 2

x = 10;
y = 5;

x *= 3; // 30
y /= 3; // 1.66...
```
